# 🎴 หมู่บ้านลี้ลับ — Mura no Nazo

## 「村の謎」

---

> *"บางสิ่งที่ถูกฝังไว้... ไม่ได้ถูกฝังเพื่อให้ใครมาขุดขึ้นมา"*

---

## 📜 โครงเรื่องหลัก (Master Plot)

### 🗾 ฉากหลัง

เรื่องราวเกิดขึ้นที่ **หมู่บ้านคิริกาคุเระ (霧隠れ村)** — หมู่บ้านโบราณแห่งหนึ่งในญี่ปุ่น ที่ซ่อนตัวอยู่ท่ามกลางหุบเขาหมอก ห่างไกลจากความเจริญ กลุ่มนักศึกษา **20 คน** จากมหาวิทยาลัย พร้อมด้วย **อาจารย์ผู้ดูแล 1 คน** เดินทางมาเพื่อศึกษาวัฒนธรรมพื้นบ้านโบราณ โดยไม่รู้เลยว่า... หมู่บ้านแห่งนี้ไม่ยอมปล่อยใครออกไปง่ายๆ

---

## 👥 ตัวละคร NPC หลัก

| ตัวละคร | บทบาท | ลักษณะ |
|---|---|---|
| **ไกด์ — ฮารุโอะ (春雄)** | ผู้นำทาง / แอดมิน NPC | ชายวัยกลางคน ผมสั้น สีหน้าปกติแต่แฝงความกังวล ไม่เห็นด้วยกับความเชื่อในหมู่บ้านแต่เงียบไว้ |
| **ลุงช่างเหล็ก — นายน้อย (小次郎 / Kojiro)** | ชายลึกลับในห้องสนิม | หนวดยาว ผมรุงรัง ตีดาบทุกคืน — แท้จริงคือคนรักของหญิงผมแดง |
| **หญิงผมแดง — อาคาเนะ (茜)** | วิญญาณ / ปริศนา | ผมแดงสด นัยน์ตาสีส้ม ร่างถูกฝังในกำแพงสนิม — ปรากฏตัวในช่วงท้าย |
| **พระสง — เอ็นคาอิ (円海)** | ผู้ผนึกจิ้งจอก | พระสงฆ์โบราณที่ยังคงอยู่ในถ้ำ รอผู้มาสืบทอด |
| **อาจารย์ — ดร. มิซูกิ** | ผู้ดูแลคณะ / แอดมิน | นักวิชาการตัวยง ตาลุกวาวเมื่อเห็นของโบราณ — เสียชีวิตช่วงท้าย |

# 📅 ไทม์ไลน์เรื่องราวแบบละเอียด

---

## 🌅 วันที่ 1 — วันแรกแห่งรอยยิ้ม

### ช่วงกลางวัน

คณะเดินทางมาถึงหมู่บ้านคิริกาคุเระ บรรยากาศอบอุ่น ชาวบ้านยิ้มแย้ม ทักทายด้วยความเป็นมิตร ฮารุโอะพาทัวร์รอบหมู่บ้าน

> 🎭 **แอดมิน (รับบทชาวบ้าน):** ทักทายผู้เล่นด้วยความเป็นมิตรเกินปกติ เสนออาหาร ชวนคุย

**สิ่งที่ผู้เล่นสังเกตได้:**

```
🚪 ห้องผนังเหล็กสนิม
— ตั้งอยู่กลางหมู่บ้านอย่างโดดเด่น
— ผนังเป็นสนิมเก่าแก่ ดูแปลกตามาก
— ฮารุโอะเดินผ่านโดยไม่กล่าวถึงแม้แต่คำเดียว
— ถ้าผู้เล่นถาม → ฮารุโอะตอบสั้นๆ ว่า "โกดังเก็บของเก่าครับ ไม่มีอะไรน่าสนใจ"
```

---

### ช่วงกลางคืน — **[เหตุการณ์แรก: ห้องสนิม]**

> แอดมินในกลุ่มผู้เล่นรวมทีมชวนไปดูห้องสนิม

**การดำเนินเหตุการณ์:**

พวกเขาแอบเดินมายังห้องสนิม และปีนขึ้นไปมองจากบน...

```
🔥 ภายในห้อง:
— เต็มไปด้วยดาบ หอก อาวุธโบราณ
— เหล็กแดงสุกจากเตาถ่าน
— ชายหนวดยาว ผมรุงรัง กำลังตีดาบอย่างตั้งใจ
— เสียงค้อนกระทบเหล็กดังก้อง...
```

> *ทันใดนั้น ชายคนนั้นหยุดตี... ช้าๆ เขาหันหลังมามอง*

**ชายหนวด (นายน้อย) พูดว่า:**
> *"...พวกแกมาทำอะไรที่นี่ ไสหัวกลับไปซะ ก่อนที่ฉันจะโกรธ"*

ไล่คณะออกไปทันที ไม่อธิบายอะไรเพิ่มเติม

---

## 🌄 วันที่ 2 — ร่องรอยแห่งศาลเจ้า

### ช่วงเช้า — **[จุดเลือก #1]**

> **ผู้เล่นเลือกได้:** จะเล่าเรื่องเมื่อคืนให้อาจารย์และฮารุโอะฟังหรือไม่?

| ตัวเลือก | ผลลัพธ์ |
|---|---|
| **เล่าให้ฟัง** | ฮารุโอะหน้าเปลี่ยน กัดฟัน แต่บอกว่า "ไม่เป็นไร... ระวังตัวด้วย" อาจารย์จดบันทึกอย่างสนใจ |
| **ไม่เล่า** | ดำเนินเรื่องปกติ แต่ฮารุโอะดูเหมือนจะรู้อยู่ดี |

---

### ช่วงกลางวัน — **ท้ายหมู่บ้านและซากศาลเจ้า**

ฮารุโอะพาทีมไปยังท้ายหมู่บ้าน

```
🎋 สิ่งที่พบ:
— ไผ่ขึ้นแน่นจนมองแทบไม่เห็นทางข้างหน้า
— รูปปั้นจิ้งจอก (キツネ) เรียงรายรอบหมู่บ้าน
  นับได้มากกว่า 40 ตัว
— ซากศาลเจ้าญี่ปุ่น บนเนินเขา — สร้างไม่เสร็จ
— ฮารุโอะพาสำรวจเฉพาะรอบเชิงเขา ไม่พาขึ้นไป
```

> 🦊 **[หมายเหตุ GM]:** ผู้เล่นที่สังเกตดีๆ จะเห็นว่ารูปปั้นจิ้งจอกบางตัว... หันหน้าไม่ตรงกับที่เคยเห็นตอนเช้า

---

### ช่วงดึก — **[เหตุการณ์ที่สอง: ศาลเจ้าสร้างไม่เสร็จ]**

กลุ่มนักศึกษาสายผีแอบขึ้นไปยังศาลเจ้า

**บรรยากาศ:**
```
🌙 ความมืด... เสียงใบไม้ไหว
   ไม่มีผีแม้แต่ตนเดียว
   แต่ภายในซากศาลเจ้า...
```

**สิ่งที่ค้นพบ:**

```
📜 บันทึกกระดาษเก่า — เขียนด้วยลายมือสั่นเครือ:

"...ข้าไม่น่ารับงานนี้มาเลย
 ถ้าไม่ใช่เพราะ นายน้อย คนนั้น
 ผู้หญิงผมแดงคนนั้น... เธอยืนกรานให้สร้าง
 ข้าสร้างไม่เสร็จหรอก... ที่นี่มันไม่ยอมให้สร้าง"

🔨 อุปกรณ์ก่อสร้างถูกทิ้งไว้กลางคัน
🩸 คราบเลือดเก่าๆ บนพื้นศาลเจ้า
```

> *ทันใดนั้น... เสียงหนึ่งดังขึ้นตามสายลม*
>
> **"...พวกแกมาทำอะไรกันที่นี่ ออกไปจากที่แห่งนี้ซะ"**

---

**ระหว่างทางกลับ:**

```
🦊 รูปปั้นจิ้งจอกเพิ่มขึ้น...
   นับไม่ถ้วนแล้ว

👥 เงาผู้คนจำนวนมากยืนอยู่หน้าศาลเจ้า
   พวกเขาเป็นคนจริงๆ หรือ...?
   ก่อนที่จะมองชัด ทุกอย่างก็หายไป
```

---

## ☀️ วันที่ 3 — ความจริงในถ้ำ

### ช่วงเช้า

กลุ่มที่ขึ้นไปศาลเจ้าเล่าทุกอย่างให้ฮารุโอะและอาจารย์ฟัง

> ฮารุโอะ: *หน้าถอดสี ยืนนิ่งอยู่ครู่หนึ่ง...*
> *"ถ้าพวกแกอยากรู้ขนาดนั้น... ฉันพาไปเอง"*
> *ยกเลิกกิจกรรมทั้งหมดของวันนั้น*

---

### ช่วงบ่าย — **ถ้ำและเสาโทริอิ**

ฮารุโอะพาเดินเลยศาลเจ้าออกไป จนถึง **ถ้ำที่ซ่อนอยู่ใกล้ศาลเจ้า**

```
🏮 หน้าปากถ้ำ — มืดทึบ
   แต่ภายในมองเห็น...
   ⛩️ เสาโทริอิ ตั้งอยู่ในความมืด
```

**ฮารุโอะเล่าความจริง:**

> *"จริงๆ แล้ว... ฉันก็ไม่ได้เห็นด้วยกับคนในหมู่บ้านนักหรอก"*

```
📖 เรื่องราวที่ฮารุโอะเล่า:

หมู่บ้านแห่งนี้นับถือ โยไค จิ้งจอก และภูติผี
มาตั้งแต่โบราณกาล

เมื่อหลายสิบปีก่อน...
มีหญิงงามคนหนึ่ง — ผมสีแดงสด นัยน์ตาสีส้ม
อาศัยอยู่ในหมู่บ้านนี้
เธอไม่เชื่อเรื่องผี และต่อต้านความเชื่อของหมู่บ้าน
อย่างสุดตัว

ชายหนุ่มจากในเมือง — ลูกชายตระกูลใหญ่
เดินทางมาศึกษาวัฒนธรรม... เหมือนพวกแก
เขาหลงรักเธอ และพาเธอออกไปจากหมู่บ้าน

ปีถัดมา ทั้งคู่กลับมา
เพื่อสร้างศาลเจ้าตามที่เคยบนบานไว้...
แต่ชาวบ้านไม่ยอม

พวกเขาไปทำพิธีกับจิ้งจอก
เพื่อขัดขวางการก่อสร้าง

มีพระสงค์รูปหนึ่งที่อาสาเข้าต่อสู้
เขาวางรากฐานศาลเจ้าไว้ในถ้ำแห่งนี้
และผนึกอะไรบางอย่างเอาไว้...

ส่วนหญิงสาวผมแดง...
[ฮารุโอะหยุด กัดฟัน]
ชาวบ้านฆ่าเธอ
และฝังร่างเธอไว้ใน... กำแพงเหล็กนั้น"
```

---

### ช่วงดึก — **[เหตุการณ์ที่สาม: ภายในถ้ำ]**

นักศึกษาตัดสินใจเข้าไปในถ้ำ

```
⛩️ ภายในถ้ำ:
   ศาลเจ้าที่สมบูรณ์ — ตั้งอยู่ในความมืด
   เทียนจุดอยู่เองโดยไม่มีใครจุด
   และ...
```

**พบกับ พระเอ็นคาอิ — พระสงค์ในตำนาน**

> *"พวกเจ้า... มาถึงแล้ว"*
> *"ข้ารู้ว่าพวกเจ้ามา เพราะพวกเจ้ารู้แล้ว"*

ผู้เล่นถามว่า รู้อะไร?

> *"จงตามหากริช... ข้าซ่อนไว้ในถ้ำแห่งนี้*
> *มันคืออาวุธที่สามารถจัดการจิ้งจอกได้*
> *แต่จงระวัง... พวกมันรู้ว่าพวกเจ้าอยู่ที่นี่แล้ว"*

---

### ⚔️ **[การต่อสู้: ปีศาจจิ้งจอกในถ้ำ]**

```
🦊 จิ้งจอกปรากฏตัวจากความมืด
   ร่างกึ่งคนกึ่งสัตว์ ตาเหลืองลุก
```

**ระบบความสามารถพิเศษของผู้เล่น:**

| ความสามารถ | คำอธิบาย | ผลในการต่อสู้ |
|---|---|---|
| 👁️ **มองเห็นในที่มืด** | บางคนมองเห็นได้ชัดแม้ไม่มีแสง | ระบุตำแหน่งจิ้งจอกได้ก่อน |
| 🩸 **เลือดมาก** | ร่างกายทนทานผิดปกติ | รับดาเมจได้มากกว่าคนอื่น |
| 💪 **โจมตีแรง** | แรงกายเกินมนุษย์ปกติ | ดาเมจสูง แต่ช้ากว่า |
| 🧠 **ไหวพริบสูง** | อ่านสถานการณ์เก่ง | สามารถหาจุดอ่อนของจิ้งจอกได้ |
| 🔮 **รู้สึกได้** | รับรู้พลังเหนือธรรมชาติ | เตือนภัยล่วงหน้าได้ |

---

**ผลลัพธ์:** ทีมหาและได้กริชมา **2 เล่ม**

```
🗡️ กริชเล่มแรก  — อาวุธของพระเอ็นคาอิ
📖 กริชเล่มที่สอง — นักศึกษาคนหนึ่งหยิบขึ้นมา...

"เล่มนี้... ตราประจำตระกูลฉัน"
→ เขาคือลูกหลานสายเลือดของพระเอ็นคาอิ
```

---

**ระหว่างทางกลับ:**
```
💡 หมู่บ้านสว่างผิดปกติ...
   แต่ทุกคนเหนื่อยเกินกว่าจะสนใจ
```

---

### ตลอดทั้งคืน — **การหลอกหลอน**

```
😰 ไม่มีใครนอนหลับได้
   เสียงเคาะประตู... ไม่มีใครอยู่
   เงาใต้ประตู... ขยับ
   เสียงกระซิบชื่อแต่ละคน...
   รูปปั้นจิ้งจอกหน้าห้อง — มาจากไหน?
```

## 🔥 วันที่ 5-6 — **คืนสุดท้าย** *(เล่นยาวใน 1 วันจริง)*

### ช่วงเช้าวันที่ 5

กลุ่มนักศึกษาเล่าทุกอย่างให้ฮารุโอะและอาจารย์ฟัง

**อาจารย์เห็นกริช:**

```
✨ ดร.มิซูกิตาลุกวาว
   เดินเข้ามาดูอย่างตื่นเต้น ไม่อาจซ่อนอารมณ์ได้

"นี่คือ... กริชยุคเอโดะ ราว 400 ปีก่อน
 ด้ามทำจากงาช้างและทอง ฝักแกะด้วยลายจิ้งจอกเก้าหาง
 แต่ที่พิเศษกว่าคือ... ใบมีดนี้"

[ยกขึ้นมาดูในแสง]

"มันไม่ใช่เหล็กธรรมดา
 ดูสีของมัน — น้ำเงินอมม่วง
 นี่คือ 'เหล็กผีดำ' (黒霊鉄 / Kuro Reitetsu)
 โลหะที่ตีจากอุกกาบาตที่ตกในยามดึก
 ตำราโบราณบอกว่า... สิ่งเหนือธรรมชาติ
 ไม่อาจทนต่อมันได้"

[หันมายิ้มที่กลุ่มนักศึกษา]

"นี่คือการค้นพบทางวิชาการครั้งยิ่งใหญ่ที่สุด
 ในรอบศตวรรษเลยนะ"
```

**ฮารุโอะ:** ดูยิ้มอย่างโล่งใจ แต่ซ่อนบางอย่างไว้ในสายตา

> *"วันนี้พวกคุณพักผ่อนเถอะ พรุ่งนี้เช้า เราออกเดินทางกลับกัน"*

---

### ช่วงบ่าย — **[บรรยากาศเปลี่ยน]**

```
😶 ชาวบ้านที่เคยยิ้มแย้ม...
   หันหลังให้เมื่อเห็นพวกเขา
   เด็กๆ ถูกดึงกลับบ้านเมื่อมองมาทาง
   ร้านค้าปิดตัวก่อนเวลา
   มีคนยืนมองจากหน้าต่างมืดๆ
```

> 💡 **[GM แนะนำ — เพิ่มเติมตรงนี้]:**
> - ผู้เล่นที่มีความสามารถพิเศษ "รู้สึกได้" จะสัมผัสได้ว่ามีพลังบางอย่างสะสมอยู่
> - ในตอนเย็น จะมีชาวบ้านคนหนึ่งแอบมาเตือนเป็นการส่วนตัว — กระซิบว่า *"พวกคุณอย่านอนคืนนี้... แค่นั้นพอ"* แล้วหนีไป
> - บางคนอาจสังเกตว่า ฮารุโอะหายตัวไปครึ่งวัน

### ตกดึก — **[ไฟลุกที่หน้าบ้าน]**

```
🔥 เปลวไฟลุกโชนหน้าที่พัก
   ชาวบ้านถือคบเพลิง ยืนล้อม
   บางคนถืออาวุธ
   ตาพวกเขา... ว่างเปล่า
   เหมือนไม่ใช่คนอีกต่อไป
```

> **ผู้นำชาวบ้านตะโกน:**
> *"มอบกริชนั้นมา... และพวกแกจะออกไปได้โดยสวัสดิภาพ"*

### 📖 **[การค้นพบ: หนังสือในถ้ำ]**

นักศึกษาที่กลับไปถ้ำตอนเช้าพบหนังสืออีกเล่ม

```
📚 เนื้อหาในหนังสือ:

หน้า 1:
"จงมอบกริชแก่ผู้ถูกเลือก
 ผู้ถูกเลือกคือผู้ที่ใช้จี้หยกแควน
 สายเลือดของผู้พิทักษ์"

→ นักศึกษาคนที่มีจี้หยก = ผู้ถูกเลือก?
→ แต่... ลุงนายน้อยก็มีจี้หยกเหมือนกัน
```

```
หน้า 2:
"ชายผู้ตีเหล็กในคืน
 เขาตีเพื่อแก้แค้นให้ภรรยา
 ทุกดาบที่ตีคือทหารในกองทัพของเขา
 เขาปิดบังเพราะไม่ต้องการให้หมู่บ้านรู้

 และในกำแพงสนิมนั้น...
 มีร่างของเธอ — หญิงผมแดง
 ภรรยาของเขา"
```

> 💡 **ความจริงที่เฉลย:** ลุงนายน้อย = คนรักของหญิงผมแดง เขาสร้างกองทัพดาบเพื่อแก้แค้นหมู่บ้าน

### ⚔️ **[ฉากสู้ครั้งสุดท้าย]**

กลุ่มนักศึกษาหนีเข้าไปยัง **ห้องผนังเหล็กสนิม**

```
🚪 ประตูเหล็กกั้นชาวบ้านไว้ได้
   แต่ด้านนอก...

🦊 จิ้งจอก — ปรากฏตัวในร่างเต็ม
👥 โยไค — ออกมาจากความมืด
   ทุกอย่างมุ่งมาที่กริช
   พวกมันต้องการทำลายมัน
```

**ผู้เล่นต้องเลือก:**
- แบ่งกลุ่มสู้
- ใช้ความสามารถพิเศษ
- ใช้กริช 2 เล่มอย่างไร

### 🗡️ **[ลุงนายน้อยเข้าช่วย]**

```
💨 ทุกอย่างดูสิ้นหวัง...
   จิ้งจอกเข้าจนตัว

ทันใดนั้น — เสียงดาบ
ลุงนายน้อยบุกเข้ามา ดาบอยู่ในมือ

"ออกไปจากพวกเขา!"

ต่อสู้อย่างดุเดือด...
จนกระทั่ง จิ้งจอกแปลงร่าง...
```

> **จิ้งจอกแปลงเป็น... หญิงสาวผมแดง**
>
> *"...นายน้อย"*
>
> ลุงนายน้อยหยุดนิ่ง มือสั่น
>
> ร่างหญิงสาวหันมายังกลุ่มนักศึกษา
> *"พวกคุณ... หนีไปได้เลย"*
>
> **ลุงนายน้อยถูกจิ้งจอกสังหารในที่สุด**

### 🗺️ **[ทางออก]**

```
อาจารย์ดร.มิซูกิ เสียชีวิตแล้ว
(ออกไปคุยกับชาวบ้านคนเดียว — คนแรกที่จากไป)

แต่เขาทิ้งแผนที่ไว้
แผนที่ที่วาดเส้นทางออกจากหมู่บ้าน
```

กลุ่มนักศึกษาฝ่าฟันออกมาจนถึง **ปากทางออกหมู่บ้าน**

### 🔴 **[จุดเลือกสุดท้าย — ปริศนาสุดท้าย]**

```
👩‍🦰 หญิงสาวผมแดงยืนอยู่หน้าประตูออก

"หยุด..."

"นี่ไม่ใช่ทางออกที่แท้จริง"
"เชื่อฉัน... ทางออกจริงๆ อยู่ที่..."
[เธอบอกทิศทางที่แตกต่าง]
```

**ปัญหา:** ทุกคนเห็นกับตาตัวเองว่าจิ้งจอกแปลงร่างเป็นเธอได้

---

### 🎭 **[จุดแตกหัก — สองตัวเลือก]**

> **เลือกที่ 1: เชื่อหญิงผมแดง**
> ตามเธอไปยังทางออกที่แท้จริง
> → **รอด** ออกจากหมู่บ้านได้

> **เลือกที่ 2: ไม่เชื่อ — ใช้กริชข่าเธอ**
> กริชสัมผัสร่างเธอ...
> เวลาย้อนกลับ
> → **วนกลับไปวันแรก** ติดอยู่ในหมู่บ้านตลอดกาล

---

## 💡 **[วิธีแยกแยะ — ให้ผู้เล่นใช้เบาะแส]**

```
🔍 เบาะแสที่ผู้เล่นสะสมมา:

1. จิ้งจอกแปลงร่างเป็นเธอ → แต่ไม่สามารถพูดชื่อ "นายน้อย" ได้
   (หญิงสาวแท้จริงเรียกเขาด้วยชื่อเล่นส่วนตัว)

2. ตาของจิ้งจอก = สีทอง
   ตาของหญิงสาวแท้ = สีส้ม

3. กริชจะสั่นเมื่ออยู่ใกล้จิ้งจอก
   ถ้ากริชไม่สั่น... เธอคือตัวจริง

4. เธอรู้เรื่องห้องสนิมที่ไม่มีใครควรรู้
```
