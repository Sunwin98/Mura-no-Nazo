## หมวด 3 — ทัศนคติต่อ Roleplay

**3.1** คุณนิยาม "Roleplay ที่ดี" ว่าคืออะไร อธิบายด้วยคำพูดของตัวเอง

```
ตอบ:
```

**3.2** ถ้าผู้เล่นคนหนึ่งทำอะไรบางอย่างที่ "เสียบรรยากาศ" (เช่น break character, พูดนอกเรื่องในช่วง RP จริงจัง) คุณจะจัดการยังไง

```
ตอบ:
```

**3.3** ในมุมมองของคุณ เส้นแบ่งระหว่าง "ดราม่าในเนื้อเรื่อง" กับ "ดราม่านอกเรื่อง" คืออะไร และคุณจะป้องกันไม่ให้มันปนกันได้อย่างไร

```
ตอบ:
```

**3.4** ถ้าตัวละครของผู้เล่นคนหนึ่ง "ตาย" ในเรื่อง แต่ผู้เล่นคนนั้นโกรธและไม่ยอมรับผล คุณจะรับมือยังไง

```
ตอบ:
```

**3.5** คุณคิดว่า RP ที่ดีควรเน้น "ความสนุก" หรือ "ความสมจริง/สมเหตุสมผล" กว่ากัน และทำไม

```
ตอบ:
```

## หมวด 4 — ความเข้าใจในเซิร์ฟเวอร์นี้

**4.1** คุณรู้จักเซิร์ฟเวอร์หมู่บ้านลี้ลับมาจากไหน และรู้สึกอะไรกับคอนเซปต์ของมันครั้งแรกที่เห็น

```
ตอบ:
```

**4.2** เซิร์ฟเวอร์นี้มีธีมญี่ปุ่นโบราณ-ลึกลับ และเน้นเนื้อเรื่องเป็นหลัก คุณมีความรู้หรือความสนใจในธีมนี้แค่ไหน (ไม่ต้องเก่ง แค่บอกตรงๆ)

```
ตอบ:
```

**4.3** ถ้าคุณต้องอธิบายบรรยากาศที่อยากให้ผู้เล่นรู้สึกเมื่อเล่นเซิร์ฟเวอร์นี้ คุณจะบอกว่ายังไง (ใช้คำไม่เกิน 3 คำ)

```
ตอบ:
```

**4.4** มีอะไรในเซิร์ฟเวอร์ที่คุณอยากพัฒนาหรือเพิ่มเติมมากที่สุด ถ้ามีโอกาส

```
ตอบ:
```

## หมวด 5 — สถานการณ์สมมติ

> อ่านสถานการณ์แต่ละข้อแล้วบอกว่าคุณจะทำอะไร ไม่มีคำตอบที่ถูกที่สุด แต่ต้องการดูว่าคุณคิดยังไง

**5.1** คืนหนึ่งระหว่าง event สำคัญ ผู้เล่น 2 คนทะเลาะกันจริงๆ นอกเรื่อง (ไม่ใช่ในบทบาท) จนบรรยากาศแตก คุณเป็นแอดมินที่อยู่ในช่วงนั้น คุณจะทำอะไรก่อนเลย

```
ตอบ:
```

**5.2** คุณได้รับข้อมูลว่ามีผู้เล่นคนหนึ่งบอกคนอื่นว่าเขียน "Lore" ของเซิร์ฟเวอร์เองเองทั้งหมด ทั้งที่ไม่จริง คุณจะทำอะไร

```
ตอบ:
```

**5.3** ผู้เล่นคนหนึ่งมาบอกคุณแบบส่วนตัวว่าเขารู้สึก "ถูกเพื่อนในทีมแอดมิน" ปฏิบัติไม่ยุติธรรม คุณไม่รู้ว่าเรื่องนี้จริงหรือไม่ คุณจะเริ่มต้นยังไง

```
ตอบ:
```

**5.4** คุณมีไอเดียใหม่ที่คิดว่าดีมากสำหรับเซิร์ฟเวอร์ แต่คนอื่นในทีมส่วนใหญ่ไม่เห็นด้วย คุณจะทำอะไรต่อ

```
ตอบ:
```

**5.5** ถ้า GM บอกว่าตัวละครของผู้เล่นคนหนึ่งควรจะตายในฉากนี้ตามเนื้อเรื่อง แต่ผู้เล่นคนนั้นเป็นเพื่อนของคุณ คุณจะทำยังไง

```
ตอบ:
```

## หมวด 6 — ความพร้อมและความมุ่งมั่น

**6.1** คุณสามารถทุ่มเวลาให้กับเซิร์ฟเวอร์ได้กี่ชั่วโมงต่อสัปดาห์โดยประมาณ (ตอบตามจริง ไม่ใช่ตอบให้ดูดี)

```
ตอบ:
```

**6.2** มีช่วงเวลาไหนบ้างที่ไม่ว่างหรือต้องหายไปนาน (เช่น สอบ, งาน, เรียน) ช่วง 3 เดือนข้างหน้า

```
ตอบ:
```

**6.3** ถ้าเซิร์ฟเวอร์ต้องการให้ทำงานบางอย่างที่น่าเบื่อหรือซ้ำซาก (เช่น รีวิวใบสมัครผู้เล่น, ตอบ DM ซ้ำๆ) คุณจะรับมือกับมันยังไง

```
ตอบ:
```

**6.4** ถ้าถูกปรับตำแหน่งหรือถอดออกจากทีมโดยมีเหตุผลที่ชัดเจน คุณคิดว่าตัวเองจะรับมันได้ไหม และทำไม

```
ตอบ:
```

**6.5** เหตุผลหลักที่คุณอยากเป็นแอดมินของเซิร์ฟเวอร์นี้คืออะไร (ตอบตรงๆ ไม่ต้องสวยหรู)

```
ตอบ:
```
